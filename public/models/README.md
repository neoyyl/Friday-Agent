# VRM 模型目录

将你的 VRM 模型放在此目录下，命名为 `aira.vrm`。

## 快速获取 VRM 模型

### 方案 1：VRoid Studio（推荐，免费）

1. 下载 [VRoid Studio](https://vroid.com/studio)
2. 选择模板创建角色
3. 调整外观（发型、服装、面部）
4. 导出为 VRM 格式
5. 重命名为 `aira.vrm` 放入此目录

### 方案 2：VRoid Hub（免费模型）

1. 访问 [VRoid Hub](https://hub.vroid.com)
2. 搜索免费模型
3. 下载 VRM 文件
4. 重命名为 `aira.vrm` 放入此目录

### 方案 3：ReadyPlayerMe

1. 访问 [ReadyPlayerMe](https://readyplayer.me)
2. 创建头像
3. 导出 VRM 格式

## 模型要求

- 格式：VRM 0.x 或 VRM 1.0
- 面数：< 100,000
- 纹理：1024x1024 或 2048x2048
- 骨骼：标准人形骨骼

## 测试

如果没有 VRM 模型，组件会自动显示一个占位符（蓝色线框球体）。

放置模型后，刷新应用即可看到 3D 虚拟形象。
